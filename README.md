# Project 1: CRM Data Analysis and Interactive Dashboard Development using Python and Power BI
This project is part of my internship work, where I performed CRM data analysis using Python and Power BI.

What Did:
1. Learned basic CRM concepts like sales funnel, pipeline, metrics, and customer segmentation
2. Converted raw TXT data into CSV format
3. Cleaned data using Python (handled missing values, removed duplicates)
4. Standardized date formats using Pandas
5. Built an interactive Power BI dashboard

Dashboard Features:
1. KPI cards and sales funnel
2. Lead source and industry analysis
3. Geographic and time-based insights
4. Filters and slicers for better interaction

This project helped me understand how to clean raw data and convert it into useful business insights through dashboards.
